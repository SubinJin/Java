CSS Reset

css를 적용하더라도 모든 브라우저에 동일하게 보이는 것은 아님

각 브라우저마다 기본적으로 적용되어있는 스타일의 차이 때문인데 이것을 CSS Reset이라는 특수한 기능으로 모든 브라우저의 기본 스타일(여백 등)을 초기화 시킬 수 있음

normalize.css를 이용해서 똑같이 적용할 수 있음
