HTML5 - 문자 다루기

strong : 조금 더 강조

em : 강조하기

sup : 위 첨자로 표시

sub : 아래 첨자로 표시

ins : 밑줄로 표시(새로 추가됨)

del : 삭선으로 표시(삭제한 것)

mark : 하이라이트로 표시
